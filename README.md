**Dewin-Conference-Telegram-Bot** - это Telegram бот, который может помочь с возможностью модерирования группы, удобного информирования пользователей, а так же принесет вам в группу развлечения!

**Структура бота**
```
DewinBot/
│
├── main.py                   # Главный файл для запуска бота
├── config.py                 # Конфигурационный файл (токены, настройки)
│
├── commands/                 # Папка с командами
│   ├── __init__.py           # Позволяет использовать папку как модуль
│   ├── notifications/        # Папка с уведомлениями о входе/выходе из беседы
│   │   ├── handlers.py         # Обработчики команд/тригеров
│   │   ├── entering.py         # Уведомление о входе в беседу
│   │   └── exiting .py         # Уведомление о выходе их беседы
│   │
│   ├── admin/    	      # Папка с админ-командами
│   │   ├── handlers.py         # Обработчики команд/тригеров
│   │   ├── ahelp.py            # Список админ-команд
│   │   ├── setadmin.py         # Назначение/повышение администратора беседы
│   │   ├── delladmin.py        # Снятие/понижение администратора беседы
│   │   ├── all.py              # Упоминание всех пользователей в группе
│   │   └── sms.py              # Отправка важного сообщения всем пользователям группы в ЛС
│   │
│   ├── entertaining/         # Папка с развлекательными командами
│   │   ├── interactions        # Папка с модулями взаимодействий и действий
│   │   │   ├── handlers.py       # Обработчики команд/тригеров
│   │   │   ├── ehelp.py          # Список развлекательных команд
│   │   │   ├── hug.py            # Обнять
│   │   │   ├── love.py           # Заняться любовью
│   │   │   ├── hit.py            # Ударить
│   │   │   ├── kiss.py           # Поцеловать
│   │   │   ├── slap.py           # Дать пощечину
│   │   │   ├── dance.py          # Пригласить на танец
│   │   │   ├── highfive.py       # Дать пять
│   │   │   ├── boop.py           # Легонько ударить по носу
│   │   │   ├── headpat.py        # Погладить по голове
│   │   │   ├── compliment.py     # Сделать комплимент
│   │   │   ├── wave.py           # Помахать рукой
│   │   │   ├── wishluck.py       # Пожелать удачи
│   │   │   ├── handshake.py      # Рукопожатие
│   │   │   ├── good_morning.py   # Пожелать доброго утра
│   │   │   └── good_night.py     # Пожелать доброй ночи
│   │   │
│   │   ├── events/               # Папка с системами репутации, опыта и уровней
│   │   │   ├── handlers.py       # Обработчики команд/тригеров
│   │   │   └── reputation.py       # Система репутации
│   │   │
│   │   └── wedding/            # Папка с системой браков
│   │       ├── handlers.py       # Обработчики команд/тригеров
│   │       ├── wedding.py        # Предложить заключить брачный договор
│   │       ├── divorces.py       # Предложить расторгнуть брачный договор — Развестись
│   │       └── marriage.py       # Посмотреть список бракосочетаний в беседе
│   │
│   ├── public/               # Папка с общедоступными командами
│   │   ├── handlers.py          # Обработчики команд/тригеров
│   │   ├── reg.py               # Регистрация бота в группе (создание таблицы в БД)
│   │   ├── help.py              # Список команд бота. Помощь по работе бота
│   │   ├── admins.py            # Показывает список модераторов беседы
│   │   ├── top.py               # Топ пользователей (в беседе) по кол-во сообщений
│   │   ├── info.py              # Показывает информацию о боте
│   │   ├── profile.py           # Информация (профиль) о себе
│   │   └── quit.py              # Система выхода из беседы с помощью бота
│   │
│   ├── private/              # Папка с командами доступными только в ЛС бота
│   │   ├── handlers.py          # Обработчики команд/тригеров
│   │   ├── keyboards.py         # Кнопки в ЛС бота
│   │   └── start.py             # Включение уведомлений в боте (для команды /sms)
│   │
│   └── developers/           # Папка с командами доступными только разработчикам
│       ├── handlers.py          # Обработчики команд/тригеров 
│       └── Вам знать не надо*
│
├── support/
│   └── src/
│       ├── handlers/
│       │   ├── __init__.py
│       │   ├── tickets.py
│       │   └── users.py
│       ├── keyboards/
│       │   ├── __init__.py
│       │   └── tickets.py
│       ├── database/
│       │   ├── __init__.py
│       │   └── models.py
│       └─── middlewares/
│           └── __init__.py
│
├── database/                 # Папка для работы с базой данных
│   ├── __init__.py             # Позволяет использовать папку как модуль
│   ├── database.py             # Логика работы с БД
│   ├── db.sqlite3              # Файл с БД
│   └── db_export.txt           # Текстовый файл с БД (для чтения)
│
├── media/                    # Папка с медиафайлами (фото, видео и т.д.)
│   ├── photos/                 # Папка с фотографиями
│   ├── gif/                    # Папка с гиф
│   └── videos/                 # Папка с видео
│
├── utils/                    # Утилиты и вспомогательные функции
│   ├── __init__.py             # Позволяет использовать папку как модуль
│   ├── loader.py               # Подгрузка dp
│   └── texts.py                # Хранилище длинных сообщений 
│
├── README.txt                # Информация о расположении файлов
└── requirements.txt          # Зависимости проекта
```

> [!IMPORTANT]
> Это мой один из первых проектов в Telegram, создавался в целях использовать локально, но позже он разросся и заботал в 83 группах, имел 350 уникальных пользователей и за все своле время обработал 85532 сообщений от пользователей.
